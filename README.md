# mac_23

Data and code associated with:

## Short-term temperature exposures optimize larval settlement and juvenile survivorship in the reef-building coral, Montipora capitata
#### Alyssa C. Varela <sup>1</sup>, Carlo Caruso <sup>1</sup>, Joshua R. Hancock <sup>1</sup>, Khalil Smith <sup>1</sup>, Kira Hughes <sup>1</sup>, Crawford Drury <sup>1</sup>

<sup>1</sup> Hawaiʻi Institute of Marine Biology, Kāneʻohe, HI   

The file analysis.R contains code associated with this project, including data preparation, temperature profiles, larval survivorship, larval settlement, and spat survivorship. 
Additionally, the file tpc.R contains the thermal performance curve code used in this project. These processed files are available in the data directory.
